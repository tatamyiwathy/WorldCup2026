# WorldCup2026
A single-file JSON dataset containing the complete match results and details of the FIFA World Cup 2026 in Japanese.


## Data Structure

```JSON
  {
  "date": "2026-06-11",
  "home": "メキシコ",
  "home_score": 2,
  "away": "南アフリカ",
  "away_score": 0,
  "notes": null,
  "group": "グループA",
  "stage": "グループステージ",
  "studium": "エスタディオ・アステカ・メキシコシティ"
}
```
